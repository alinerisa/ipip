# ipip direct tunnel
direct ipip tunnel script

Install
<code>bash <(curl -Ls https://raw.githubusercontent.com/alinerisa/ipip/main/install.sh)</code>

برای نصب اسکریپت رو کپی کنید

گزینه 1 . برای نصب و افزودن سرور به سرور های تانل هست

گزینه 2 . نمایش وضعیت تانل

گزینه 3 . اضافه کردن پورت به تانل ( برای تک سرور)

گزینه 4 . پاک کردن تانل

گزینه 5 . خروج از اسکریپت
